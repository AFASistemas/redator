Como usar

  1. Abra o Claude e mantenha uma única conversa aberta para os 3 prompts
  2. Toque em Copiar no Prompt 1, cole no Claude, preencha os campos em [COLCHETES] e envie
  3. Em seguida, copie o Prompt 2 e cole na mesma conversa
  4. Por fim, copie o Prompt 3 para auditar o material antes de apresentar

Lembrete importante

Os 3 prompts rodam na MESMA conversa do Claude, em sequência. Não abra uma conversa nova para cada prompt — o Prompt 2 e o Prompt 3 dependem do resultado do anterior. Se você fechar a aba do Claude, abra de novo o mesmo chat antes de continuar.
